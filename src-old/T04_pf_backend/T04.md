# Note on `tsconfig.json`

- Once we use `outDir` option any non-`./src` folder needs to be included, including the root.

```json
{
  "compilerOptions": {
    "outDir": "./dist",
    }
  },
  "include": ["./*", "./src/**/*", "./db/**/*"]
}
```
